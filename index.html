Projects

Detection and Classification of Lung Cancer by Using Convolution Neural Networks (CNN) on CT Images

Master Diploma Thesis

The goal was to build a model with high accuracy in predicting whether the nodule in the image is malignant or benign, using less time and computational power than other well-known Convolutional Neural Networks (CNNs). The dataset I use is LUNA16 which provides 888 low dose CT lung images with some annotations in csv files. The medical images are in .mhd + .raw format. The library I used to read the files is SimpleITK and Pandas for csv. There were a total of 551065 annotations. I preprocessed the data to create small 50x50 2D images with nodules as input to my compact CNN. I also created 3D nodule images to observe if my compact CNN works better with more spatial information as input. Finally, I compared the classification accuracy of each model, the runtime, and the computational power required by each model.

Library Used: SimpleITK, Pandas, Tensorflow, h5py

Digital Image Watermarking

Digital Image Processing and Analysis Subject

Initially, we needed to import the image of a flower into Matlab. Next, we needed to create a random watermark using a pseudo-random function of the same dimensions as our image. After that, we had to split the image and watermark into square blocks of size 16x16 and 32x32. We should then insert the watermark into the image a) in the spatial domain b) in the frequency domain (via DCT transformation). The Embedding Algorithm used was the following: The watermark is inserted into the image by matching image block with watermark block, but with a pseudo-random watermark block each time. That is, the block chosen each time will depend on a pseudo-random generator that generates a pseudo-random sequence that defines the order of watermark blocks. Finally, we needed to test the robustness of the watermark after some kinds of attacks: namely Gaussian noise attack and geometric attacks.

Program used: Matlab
