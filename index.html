Projects

Detection and Classification of Lung Cancer by Using Convolution Neural Networks (CNN) on CT Images

Master Diploma Thesis

The goal was to build a model with high accuracy in predicting whether the nodule in the image is malignant or benign, using less time and computational power than other well-known multi-user Cellular Neural Networks (CNNs). The dataset I use is LUNA16 which provides 888 low dose CT lung images with some annotations in csv files. The medical images are in .mhd + .raw format. The library I used to read the files is SimpleITK and Pandas for csv. There were a total of 551065 annotations. I preprocessed the data to create small 50x50 2D images with nodules as input to my compact CNN. I also created 3D nodule images to observe if my compact CNN works better with more spatial information as input. Finally, I compared the classification accuracy of each model, the runtime, and the computational power required by each model.

Library Used: SimpleITK, Pandas, Tensorflow, h5py
